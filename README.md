# Holy Grail Layout

Este é um layout básico "Holy Grail" criado para ser usado como base em projetos futuros.

## Descrição

Este projeto é parte da minha jornada de aprendizado front-end com o 'The Odin Project'. É uma implementação simples em HTML e CSS do clássico layout de página web "Holy Grail".

## Tecnologias Utilizadas

*   HTML5
*   CSS3 (Grid Layout)

## Como Usar

Basta clonar este repositório e usar o `index.html` e `styles.css` como ponto de partida para o seu próprio projeto.
